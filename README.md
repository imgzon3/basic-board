## 프로젝트 소개 및 목적
- **basic-board**는 `Spring boot`를 학습하기 위한 게시판 프로젝트입니다.
- 구조를 설계하고 구현해 보는 것이 목적입니다.
    - `Spring MVC`패턴을 통해 웹페이지를 만들어봅니다.
    - 동시에 `HTTP API`를 만들어봅니다.
- 다양한 기술 스택을 학습하고, 적용합니다.
